# Full Stack Web Development Projects

This repository contains three full-stack web applications developed as part of a Java full-stack development course.

## Projects Overview

### 1. AGRIEQUIPMENTS
An online platform for renting and booking agricultural equipment. Farmers and users can browse, book, and manage equipment rentals.

**Features:**
- User registration and login
- Equipment listing and search
- Booking system
- Admin dashboard for managing equipment and users

### 2. HomeServiceProvider
A platform connecting customers with home service providers (plumbers, electricians, etc.).

**Features:**
- Service provider registration
- Customer booking
- Review and rating system
- Email OTP for authentication

### 3. LifeDrop
A blood donation management system for hospitals, donors, and blood banks.

**Features:**
- Donor registration
- Blood inventory management
- Hospital requests
- Emergency blood requests
- Camp management

## Technologies Used
- **Backend:** Java, Spring Boot, Maven
- **Frontend:** React, Vite
- **Database:** SQLite
- **Authentication:** JWT, OTP
- **Other:** REST APIs, File uploads

## Project Structure
Each project follows a similar structure:
- `BackEnd/` or similar: Spring Boot application
- `FrountEnd/` or `src/`: React frontend
- Database files and uploads

## Getting Started
1. Clone the repository
2. For each project, navigate to the backend folder and run `mvn spring-boot:run`
3. Navigate to the frontend folder and run `npm install` then `npm run dev`
4. Access the application at the provided URLs
